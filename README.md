Defense Display

Unityで制作した、捕球確率やボールと守備の範囲を可視化する、プロトタイプです。 





機能・特徴

特徴1: クリックでボールを飛ばすパワーと飛ばすところを変えれる



特徴2: 右クリックでリプレイできる（最初から）





工夫した点: \[確率ラグと可視化の範囲を変える]



🕹️ 操作方法


マウスなどのカーソル操作できるもので場所を指定


捕球後右クリックでリプレイ



🛠️ 使用技術

エンジン: Unity 



言語: C#



OS: Windows 11





🚀 インストール・実行方法

このリポジトリをクローンまたはZipでダウンロードします。



Unity Hubからプロジェクトを追加して開きます。



Scenes フォルダ内の Defense Display を開いて再生してください。
